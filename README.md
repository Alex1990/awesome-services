# awesome-service

A collection of the services, including IaaS, PaaS, SaaS.

**Goal**: Help application constructors choosing a service quickly.

**TOC**

* Cloud Computing Platform
* Analytics

* CI and CD
* CMS
* CRM
* Design
* Domain
* Office
* Email
* Fonts
* Forms
* Monitoring and Alert
* Message
* Payment
* Search
* Security
* Test
* Todos and Collaboration
* Translation
* Misc

Some factors:

* Location
* Open Source
* Selfhosted
* Free plan
