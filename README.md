# awesome-service

A collection of the services, including IaaS, PaaS, SaaS.

**Goal**: Help application constructors choosing a service quickly.

**TOC**

* Cloud Computing Platform
* Analytics
* CDN
* Chat
* CI and CD
* CMS
* CRM
* Design
* Domain
* Office
* Email
* Font
* Form
* Monitoring and Alert
* Message
* Payment
* Project Management
* Search
* Test
* Translation
* Misc

Some factors:

* Location
* Open Source
* Selfhosted
* Free plan
